The trials and tribulations of this website.

  • Lance Carter
  • Web Design

Hello! This website has been quite the journey. I've had multiple iterations over time, and I am pretty happy with this one as a good foundation to build on. It originally started as me wanting to create this whole website from scratch. I took to YouTube and the other resources to find pieces of things that I liked and bring them together into a frankenstein portfolio. Over time I came to realize that being able to code an entire website from scratch is certainly something that would demonstrate my skills, but isn't necessary.

My journey eventually led me to bootstrap. Boostrap was perfect for me in terms of creating a nice base layer. It was nice to get to dive into what kind of tools can be used for creating some of the basic pieces of websites. Implementing bootstrap was anything but hard. I will say I eventually want to manipulate the website in various ways that make it more and more my own over time. Eventually my goal would be to get this base layer to be almost completely unrecognizable as I start to add and change more.

One important lesson I learned from this was trying to make the goal of creating a minimum viable product. I can work on this day after day, but if I don't get anything out there then all of my work might as well be for nothing. Creating an MVP for the various projects that I work on is most certainly going to become a routine for me. I can always build upon something once it is out there.

One thing I can attribute a lot of my success and growth in web development to is my time at State Farm. I worked on a team where we worked on web applications that were used by State Farm agents to help run their business. Through that I was able to get some full-stack experience which has ultimately led to me being easily capable of building this and making various changes to it that will eventually lead this site looking more and more representative of myself.

This site was also the first project that I started using things like Jira to help manage my work flow. Throughout school and work I have used various tools to help manage the work flow of the teams I've been on, but Jira is simple enough that it meets my needs for helping keep me on track in my development.

Overall I am really proud of myself for getting this out there and the progress that I have made. This was a great project to help show off what I've been working on and get myself more out there. Changes soon to come!
